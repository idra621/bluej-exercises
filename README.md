# BlueJ Java Exercises

Exercises and mini-projects from the book **“Java lernen mit BlueJ”**.

This repository focuses on building a solid foundation in:

- classes & objects  
- methods and parameters  
- constructors  
- fields  
- loops  
- arrays & ArrayList  
- basic object-oriented programming

## Purpose
These exercises are used to strengthen Java fundamentals before moving on to larger projects and algorithmic problem-solving.
